To use your Canva designs with this system, follow these exact steps to turn a static design into a fillable template:

1. Design in Canva
Create your certificate in Canva as usual.
Leave blank spaces where the recipient's name, event name, and date should go.
Export: Click Share > Download and select PDF Print for the highest quality.
2. Add Fillable "Form" Fields (Crucial Step)
Canva doesn't export "fillable" fields directly, but you can add them in 60 seconds using a free online tool like Sejda PDF Editor or PDFescape:

Upload your Canva PDF to the editor.
Select the Text Field tool.
Draw a box over the area where the Name should appear.
RENAME THE FIELD: This is the most important part. Right-click the box (or check "Field Name") and set it to:
recipient_name (for the person's name)
event_name (for the event title)
issue_date (for the date)
Style it: Set the font size and alignment (Center) in the PDF editor so it matches your Canva design.
Download the edited PDF.
3. Upload to Your System
Go to http://localhost:8080/ui.
Select your event.
Click "Choose File" and upload the PDF you just prepared.
Done! Now, whenever you type a name and click "Generate," the system will "merge" that name perfectly into your Canva design.
Summary of Field Names
Field Goal	Exact Name to Use in PDF Editor
Recipient Name	recipient_name
Event Name	event_name
Issue Date	issue_date
Pro-Tip: If you have Adobe Acrobat Pro or LibreOffice Draw, you can also add these "AcroForm" fields there under the "Forms" or "Form Controls" menu!
